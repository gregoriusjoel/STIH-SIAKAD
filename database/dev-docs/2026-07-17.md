# Laporan Perubahan Sistem SIAKAD - 17 July 2026

Laporan ini mencatat seluruh pembaruan sistem dan perubahan fungsional yang diterapkan hari ini.

---

## 🚀 Rangkuman Pembaruan Hari Ini

*   **[Penyatuan Database]** Penggabungan tabel `kelas` ke dalam `kelas_mata_kuliahs` demi efisiensi dan konsistensi relasi jadwal, KRS, dan nilai.
*   **[Perbaikan Query & Model]** Refaktorisasi query KRS, penjadwalan, dan detail kelas di seluruh modul (Admin, Dosen, Mahasiswa) agar mengarah ke kolom baru `kelas_mata_kuliah_id` menggantikan `kelas_id`.
*   **[Pembersihan Berkas Usang]** Penghapusan 7 Artisan Console Command yang sudah kadaluarsa pasca-migrasi data kelas dan relasi orang tua (FixParentDataCommand, FixKelasDosen, dll.).
*   **[Peningkatan Performa (N+1 Query)]** Optimasi pemuatan halaman nilai tugas dosen dengan menghitung total mahasiswa di Controller alih-alih melakukan query inline di dalam loop view Blade.
*   **[Peningkatan Sistem Audit Log]**
    *   Integrasi pustaka `jenssegers/agent` untuk memecah informasi browser, OS, dan tipe perangkat (desktop/mobile/tablet).
    *   Penambahan filter log audit berdasarkan Role Aktor, IP Address, dan rentang tanggal di Admin Control Panel.
    *   Desain ulang UI Log Audit untuk menampilkan kolom identitas perangkat secara inline dan ringkasan sesi lengkap.
