# Laporan Perubahan Sistem SIAKAD - 24 July 2026

Laporan ini mencatat seluruh pembaruan sistem dan peningkatan desain antarmuka yang berhasil diimplementasikan hari ini.

---

## 🚀 Rangkuman Pembaruan Hari Ini

*   **[Perbaikan Error]** Memperbaiki QueryException (`Unknown column 'prodi'`) pada halaman edit Dosen PA dan aksi tambah/transfer mahasiswa bimbingan dengan mengalihkan klausa kueri ke kolom fisik `prodi_id`.
*   **[Perbaikan Error]** Mengatasi SyntaxError (`Unexpected token '{'`) pada halaman Kelas Mata Kuliah yang memblokir kinerja Alpine.js akibat hilangnya pembatas fungsi (`},`) di JavaScript.
*   **[Pembaruan Fitur]** Mengimplementasikan pembuatan otomatis (*on-the-fly*) data pertemuan baru saat Admin memasukkan tautan Zoom untuk sesi pertemuan yang belum tercatat di database.
*   **[Peningkatan Desain]** Mendesain ulang tampilan Toast Notifikasi sukses pengisian link Zoom dengan tema minimalis modern (latar belakang putih, garis tepi merah maroon `#7a1621`, dan ikon checklist hijau lembut).
*   **[Peningkatan Tampilan]** Memperbaiki tampilan Nama Kelas pada tabel Kelas Mata Kuliah agar memuat kode kelas lengkap (misal `26HK01` bukan hanya `01`) melalui optimasi eager-loading `kelasPerkuliahan` di Controller dan pembenahan *accessor* di model.
*   **[Pembaruan Fitur]** Menyelaraskan sistem Import massal (Mahasiswa, Dosen, Orang Tua) agar terintegrasi dengan fitur login `username` terbaru, lengkap dengan validasi duplikasi data ke tabel `users`.
*   **[Dokumentasi]** Menyusun modul panduan alur kerja dan logika sistem penjadwalan otomatis di `database/dev-docs/auto_generate_jadwal.md`.
